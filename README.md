# Big Red Button Web App

Foundational Express + EJS MVC app for the Big Red Button card game.

## Structure

- `src/server.js` – Express configuration, view engine, middleware, routes, and server start
- `src/routes` – Route definitions
- `src/controllers` – Request handlers
- `src/models` – Domain data placeholders (to be replaced with PostgreSQL)
- `src/views` – EJS templates and partials
- `public` – Static assets (CSS, images)
